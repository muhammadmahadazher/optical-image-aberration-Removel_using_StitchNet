# StitchNet Laboratory UI

Local React/Vite interface for the FastAPI service.

```bash
npm ci
npm run dev
npm test
```

Set `VITE_API_URL` to change the API origin; it defaults to `http://127.0.0.1:8000`. From sync folders, prefer the repository-level `python start.py` and `python scripts/frontend.py test` commands, which use a stable per-user dependency cache.
